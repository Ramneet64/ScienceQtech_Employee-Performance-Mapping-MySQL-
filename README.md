# ScienceQtech_Employee-Performance-Mapping-MySQL-
Improve employee performance evaluation by analyzing manager-provided ratings to support HR decision-making.
Tools &amp; Technology: MySQL, SQL Queries, Data Analysis
Process:
 Analyzed employee data in MySQL to determine maximum salaries, validate job roles, and ensure alignment with
organizational standards.
 Calculated bonus structures and additional expenses to assess workforce cost distribution.
 Identified skill gaps to support HR in designing targeted employee training programs.
Result: Enabled HR to finalize accurate performance mapping and optimize training plans, improving overall organizational
efficiency.
